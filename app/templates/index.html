Self-hosted cyber portal

Unifica i tuoi script OSINT / IR in una sola GUI web

Questo starter ti dà un portale Python deployabile su Proxmox, con moduli integrati, catalogo di tool per categorie, e struttura pronta per aggiungere nuovi provider senza rifare tutto da zero.

Apri Bulk AbuseIPDB Apri Investigation

Cosa include

  • Bulk lookup AbuseIPDB per liste di IP
  • Investigazione singola IP/dominio multi-provider
  • Catalogo tool categorizzato con link rapidi
  • Dockerfile e docker-compose pronti
  • Gestione chiavi via variabili d'ambiente

Moduli integrati

Bulk AbuseIPDB

Incolli una lista di IP, ottieni score, country, ISP, domain, hostname, report count e ultimo report.

IR Investigation

Input singolo IP o dominio con risultati consolidati da AbuseIPDB, Shodan, OTX, VirusTotal e WHOIS.

Come estenderlo

  1. Crea un nuovo modulo in app/services/.
  2. Aggiungi route e template dedicati oppure aggancialo all'investigation unificata.
  3. Inserisci la chiave in .env.
  4. Aggiungi il tool al catalogo in catalog.py.
